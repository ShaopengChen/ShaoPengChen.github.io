Hugo + Github Pages搭建个人网站

May 30, 2023 阅读时间：1 分钟

Hugo 搭建 链接到标题

  • 环境：Ubuntu 20.04LTS
  • 平台：Github Pages

安装 Hugo 链接到标题 

$ sudo apt install hugo

创建网站 链接到标题 

$ hugo new site website

执行完成后会生成一个website文件夹

添加 Hugo 主题 链接到标题

选择一个你喜欢的Hugo主题，如hugo-coder，克隆到themes目录。

$ cd website/themes
$ git clone https://github.com/luizdepra/hugo-coder.git

创建文章 链接到标题 

$ hugo new content/posts/1.md

启动hugo服务 链接到标题 

$ hugo server

浏览器访问localhost:1313，即可查看本地搭建的网站。

Github Pages 建站 链接到标题

本地生成静态网站 链接到标题 

$ cd website
$ hugo

生成的网站文件都存放在public文件夹中

推送到GitHub仓库 链接到标题

需要先创建个人站点的GitHub仓库（yourname.github.io）,并建立远程分支main。

$ cd public
$ git init
$ git add .
$ git commit -m "descriptions"
$ git push origin main

完成之后，打开网页yourname.github.io即可查看。
